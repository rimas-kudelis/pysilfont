# Test suite
pysilfont has a pytest-based test suite.

### install the test framework:
```
python3 -m pip install pytest
```

## set up the folders:
```
python3 tests/setuptestdata.py
```

## get extra modules:
```
python3 -m pip install https://github.com/typemytype/GlyphConstruction.git
```

## run the test suite:
```
pytest
```
